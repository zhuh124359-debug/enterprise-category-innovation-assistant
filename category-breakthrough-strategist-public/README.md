# 品类破局官

品类破局官是一款面向企业老板、创始人和管理层的战略诊断助手，帮助企业从用户认知、市场分化、品牌定位和增长路径出发，重新审视自身业务的破局机会。

它适用于增长停滞、行业同质化、品牌定位模糊、新产品难以被用户理解、企业寻找第二增长曲线等场景。通过结构化诊断，它可以帮助企业判断：用户到底如何理解我们？我们是否还困在旧竞争里？有没有机会定义一个更清晰、更有增长空间的新方向？企业应该如何设计品牌、产品、渠道和起步路径？

## 适用场景

- 企业增长停滞，需要寻找新的突破方向
- 行业竞争激烈，价格战严重
- 新产品有优势，但用户听不懂、记不住
- 品牌定位模糊，无法形成清晰认知
- 企业想做第二增长曲线，但方向不明确
- 新业务、新品牌、新产品上市前诊断
- 管理层战略复盘与年度增长规划

## 核心能力

- 从用户认知出发，重新理解产品和品牌位置
- 识别市场分化机会，跳出同质化竞争
- 诊断新方向是否清晰、可理解、可传播、可起步
- 输出老板可决策的战略判断和行动建议
- 支持新业务孵化、品牌升级和增长路径讨论

## 推荐提示词

```text
我们是一个区域食品品牌，增长停滞，价格战严重，帮我重新看破局方向。
```

```text
我们有一个新技术产品，但客户听不懂价值，帮我重新设计市场表达和定位。
```

```text
我想做第二增长曲线，但不知道新业务该怎么定义，帮我做一次战略诊断。
```

## 使用边界

本 Skill 是战略诊断辅助工具，适合帮助企业梳理思路、发现盲点、设计讨论框架。它不替代真实市场调研、消费者访谈、商业尽调或管理层最终决策。
